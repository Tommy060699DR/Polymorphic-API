# Polymorphic-API
This application integrates data from the Frankfurter API with a unique banking spread calculation feature.

## Key Features:

latest_idr_rates: Latest IDR exchange rates with banking spread calculations

historical_idr_usd: Historical IDR to USD data

supported_currencies: List of supported currencies
